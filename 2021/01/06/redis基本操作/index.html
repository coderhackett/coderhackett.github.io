Redis基本操作

作者：hackett
微信公众号：加班猿

一、redis基本数据类型

  • redis保存的数据都是按照键值对来保存
  • 键（key）
  • 值（value）
  • 键的数据类型都是字符串
  • 值的类型有以下五种：
    1. 字符串（string）
    2. 哈希（hash）
    3. 列表（list）
    4. 无序集合（set）
    5. 有序集合（zset）

二、字符串（string）

添加修改字符串

  • set 键 值
    1. 如果键不存在，就是添加
    2. 如果键存在，就是修改
  • setex 键 过期时间 值
    1. 过期时间单位为秒
  • 添加多个键值对 – mset 键1 值1 键2 值2 键3 值3 ……
  • 已有值后面追加 – append 键 值

获取值

  • get 键
  • mget 键1 键2 键3 ……

删除键

  • del 键1 键2 键3

三、键命令

查找键

  • keys 键名
  • 可以支持 *

判断键是否存在

  • exists 键
  1. 如果存在，返回1
  2. 如果不存在，返回0

查看对应键的value类型

  • type 键

设置已有键的过期时间

  • expire 键 秒数

查看键过期时间

  • ttl 键
  1. 返回0，代表有效时间，单位：秒
  2. 返回-1，代表永远有效
  3. 返回-2，代表键不存在

四、哈希hash

添加值

  • hset 键 字段 值

添加多个值

  • hmset 键 字段1 值1 字段2 值2 ……

获取字段

  • hkeys 键
  1. 获取制定键的所有字段名

获取字段对应的值

  • hget 键 字段

获取多个字段的值

  • hmget 键 字段1 字段2

获取键所有字段的值

  • hvals 键

获取键所有字段和值

  • hgetall 键

删除字段

  • hdel 键 字段

删除键

  • hdel 键

五、列表list

添加值

  • lpush 键 值1 值2 ……
  1. 从列表左侧添加值
  • rpush 键 值1 值2 ……
  1. 从列表右侧添加值

插入值

  • linsert 键 before 或者 after 值 新值
  1. 在制定的值前面或者后面插入新值

获取

  • lrange 键 start stop （返回列表里指定范围的值）
  1. 索引从左侧开始，第一个值的索引为0
  2. 索引可以是负数，表示从尾部开始计数，如：-1代表最后一个值，-2代表倒数第二个值
  3. start，stop为要获取值的索引

修改

  • lset 键 索引 值
  1. 改指定索引的值

删除值

  • lrem 键 count 值
  1. count = 0 删除指定的所有值
  2. count < 0 删除从右往左指定数量的值
  3. count > 0 删除从左往右指定数量的值

六、无序集合set

添加值

  • sadd 键 值1 值2 值3 ……

获取值

  • smembers 键
  1. 获取集合的值

删除值

  • srem 键 值
  1. 删除指定的值

七、有序集合zset

添加值

  • zadd 键 score1 值1 score2 值2
  1. 添加值

获取值

  • zrange 键 start stop [withscores]
  1. start 从 0开始
  2. 最后一个索引为-1
  3. withscores代表同时获取score

通过score获取值

  • zrangebyscore 键 min max
  1. min代表score起始值
  2. max代表score结束值

通过值获取score

  • zscore 键 值
  1. 得到指定值的score

删除

  • zrem 键 值
  1. 删除指定的值

通过score删除值

  • zremrangebyscore 键 min max
  1. min是要删除score的最小值
  2. max是要删除score的最大值
